# ATS Rules

## Core Principles
- Single-column layout for ATS-safe templates.
- Plain text headings (no icons, no images).
- Consistent section labels: Summary, Experience, Education, Skills.
- Avoid tables, columns, and text boxes for ATS templates.
- Use standard fonts with strong OCR support.

## Required Fields
- Full name
- Email
- At least one skill

## Recommended Fields
- Phone number
- Location
- LinkedIn URL
- Portfolio URL
- At least one experience entry
- At least one education entry

## Content Guidelines
- Summaries <= 300 characters.
- Use bullet points for responsibilities.
- Quantify impact with metrics where possible.
- Use consistent date formats (e.g., 2021-01 or 2021).

## Keyword Strategy
- Mirror core keywords from the job description.
- Use exact matches for critical tools and titles.
- Place keywords in Summary, Experience, and Skills.
- Avoid keyword stuffing.

## Export/Print Rules
- A4 size, black-and-white output.
- Resume content only (no UI elements).
- Deterministic rendering from preview source.
