# SOLMATES Resume Builder

Production-grade resume builder with ATS-safe rendering, deterministic JD analysis, and export-ready outputs.

## Features
- Resume engine with parser, validator, renderer, and deterministic scoring.
- JD intelligence pipeline with explainable outputs.
- 10+ templates with ATS-safe defaults.
- Export to A4 PDF and DOCX.
- Auth, resume CRUD, templates, and analytics APIs.

## Quick Start
1. Install dependencies.
2. Start the API server.
3. Open `public/index.html` for the static UI or integrate the SPA.

## API
Base URL: `/api/v1`
